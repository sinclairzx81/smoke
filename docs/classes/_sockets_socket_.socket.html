Class Socket

A Web Socket like type built over RTCDataChannel. It supports bi-directional messaging between peers within a network. This socket type functions both as a Web Socket as well as a WebRTC pass-through for sending and receiving MediaStream Tracks to the sockets underlying RTCPeerConnection.
